# Day 8 笔记：相机标定为什么重要

YOLO 给你的不是机械臂坐标，而是图像坐标。

例子：

```text
bbox center = (465, 295)
```

这代表：

```text
图像第 465 个像素，第 295 个像素
```

机械臂需要的是：

```text
robot_x = 0.27 m
robot_y = -0.09 m
```

所以要做标定：

```text
pixel_x, pixel_y -> robot_x, robot_y
```

## 标定点是什么

你在桌面上选几个已知点。

比如：

```text
图像像素点        真实机械臂坐标
(100, 100)  ->   (-0.40,  0.25)
(320, 240)  ->   ( 0.00,  0.00)
(540, 380)  ->   ( 0.40, -0.25)
```

这些配对数据叫标定点。

## 为什么用 PyTorch 演示标定

真实工程里，平面标定通常用 OpenCV 的单应性矩阵。  
但用 PyTorch 拟合一个小模型，可以让你直观看到：

```text
给很多 输入->正确答案
模型就能学到映射
```

这和机械臂策略训练是同一个思想。

## 标定误差怎么看

预测：

```text
模型算 robot_x/robot_y
```

正确答案：

```text
人工测量的 robot_x/robot_y
```

误差：

```text
两者距离差多少米
```

误差越小，标定越准。

## 今天你必须能说出来

```text
1. YOLO bbox 是像素坐标。
2. 机械臂需要机械臂坐标。
3. 标定点是像素坐标和机械臂坐标的配对。
4. 标定误差越小，抓取目标位置越准。
5. PyTorch 也可以拟合 pixel -> robot 的映射。
```
